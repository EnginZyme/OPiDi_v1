Initialise the pipettes and check the tip strategy.

Since this class is reinitialised at the start of every pipette step, it should pick up a tip if it does not have one. If mode is ECO - new tip, drop the current tip and take a new one.

Tip Strategies:

  • ECO - Never take a new tip
  • ECO - NEW TIP - Take a new tip only at the start of each new pipette step
  • STRICT - New tip for every movement
  • STANDARD - New tip per source labware
Parameters
pipetteThe pipette object to manage.
